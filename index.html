What is a computer network?

A computer network is a type of network that allows separate computers to send and receive data, thus enabling many forms of communication between them. An example of a computer network is the internet.

Protocols and Algorithms

Using many different protocols and algorithms, devices connected in a network are able to send and receive data. Here are some examples of internet protocols:

  • Transmission Control Protocol (TCP)
  • Internet Protocol (IP)
  • Hypertext Transfer Protocol (HTTP)
  • File Transfer Protocol (FTP)

All of these protocols have a set of rules for sending, receiving and displaying data.
